Teste da Sidebar

Redimensione a janela para testar a responsividade:

  • >1024px: Sidebar sempre visível
  • ≤1024px: Sidebar oculta, botão para abrir
  • ≤768px: Botão fixo no canto
  • ≤480px: Sidebar full-width

Instruções de Teste

1. Redimensione a janela para 768px ou menos

2. Clique no botão ☰ para abrir a sidebar

3. Verifique se a sidebar desliza da esquerda

4. Clique fora para fechar
